# 99X AI Hub — Final Landing Page

**Description**

A polished static landing page for the 99X AI Hub. This repository contains a single-page HTML site (index.html) intended as the final landing page design.


## Screenshot

Add a screenshot image named `screenshot.png` to the repository root (or adjust the path below). Once the image is added, GitHub will render it here.

![Landing Page Screenshot](./screenshot.png)


## Features

- Clean, responsive single-page landing layout (HTML only).
- Hero section, features/services overview, and call-to-action.
- Ready to host as a static site (GitHub Pages, Netlify, Vercel, etc.).


## Demo / Locally

To preview locally:

1. Clone the repo:

   git clone https://github.com/BinaryVortex/99X-AI-Hub-Final-Landing-Page.git
2. Open `index.html` in your browser, or serve it with a simple server:

   - Python 3: `python -m http.server 8000` then open `http://localhost:8000`
   - Node (http-server): `npx http-server` then open the printed URL


## Files

- `index.html` — the landing page source (HTML only).
- `README.md` — this file.


## How to add the screenshot (if you don't have one yet)

1. Take a screenshot of the landing page (recommended size around 1280×720 or wider).
2. Save it as `screenshot.png` (or `screenshot.jpg`) in the repository root or inside an `assets/` or `images/` folder.
3. If you place it in a folder, update the image path above, for example:

   `![Landing Page Screenshot](./assets/screenshot.png)`


## Hosting suggestions

- GitHub Pages: enable Pages in repository settings and set the source to the `main` branch (root). The site will be available at `https://<username>.github.io/99X-AI-Hub-Final-Landing-Page/`.
- Netlify / Vercel: drag & drop the `index.html` (or connect the repo) for instant deploy.
